Прогнозирование стоимости страховых выплат (Workers Compensation)

Описание проекта
Цель проекта — разработать модель машинного обучения, которая предсказывает итоговую стоимость страхового возмещения (UltimateIncurredClaimCost) на основе характеристик работника, обстоятельств дела и начальной оценки страхового случая.  
Проект включает интерактивное веб-приложение на базе Streamlit, позволяющее:
- загрузить и предобработать данные,
- обучить и сравнить несколько регрессионных моделей,
- визуализировать результаты и важность признаков,
- выполнить предсказание для нового страхового случая.

Датасет
Используется открытый датасет Workers Compensation из репозитория OpenML (ID 42876).  
- Количество записей: 100 000  
- Признаки: 13 входных переменных + целевая  
- Целевая переменная: UltimateIncurredClaimCost (итоговая стоимость выплаты)  

Данные загружаются напрямую через sklearn.datasets.fetch_openml, поэтому дополнительная загрузка файлов не требуется.

Установка и запуск
1. Клонируйте репозиторий
bash
git clone <ссылка-на-ваш-репозиторий>
cd workers_compensation_project
2. Создайте и активируйте виртуальное окружение (рекомендуется)
python -m venv venv
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
3. Установите зависимости
pip install -r requirements.txt
4. Запустите приложение
streamlit run app.py

Основные возможности приложения
Страница «Анализ и модель»

    Загрузка и предобработка данных

        Автоматическая загрузка датасета из OpenML.

        Преобразование дат (извлечение месяца, дня недели, задержки отчёта).

        Кодирование категориальных признаков (LabelEncoder).

        Масштабирование числовых признаков (StandardScaler).

        Разделение на обучающую (80%) и тестовую (20%) выборки.

    Обучение моделей

        Доступны четыре алгоритма: LinearRegression, Ridge, RandomForestRegressor, XGBRegressor.

        Обучение происходит в реальном времени после нажатия кнопки.

    Сравнение качества

        Таблица метрик MAE, RMSE, R² с подсветкой лучших значений.

        График «Реальные vs Предсказанные значения» для выбранной модели.

        Гистограммы распределения целевой переменной в train/test.

    Анализ важности признаков

        Горизонтальная столбчатая диаграмма для Random Forest и XGBoost.

        Позволяет понять, какие факторы сильнее всего влияют на стоимость.

    Предсказание нового случая

        Форма для ввода всех параметров (возраст, зарплата, тип травмы и т.д.).

        Автоматическое применение обученных кодировщиков и масштабировщика.

        Вывод предсказанной итоговой стоимости.

Страница «Презентация»

    Интерактивные слайды, созданные с помощью библиотеки streamlit-reveal-slides.

    Возможность настройки темы, высоты слайдов, эффекта перехода и плагинов.

    Краткое описание проекта, методов и результатов.

Используемые библиотеки

    streamlit – создание интерактивного веб-интерфейса.

    pandas, numpy – обработка и анализ данных.

    scikit-learn – предобработка, модели, метрики, разделение выборки.

    xgboost – градиентный бустинг.

    matplotlib – визуализация графиков.

    streamlit-reveal-slides – встраивание Reveal.js презентаций.

Полный список с версиями см. в requirements.txt.

Примечания

    При первом запуске датасет загружается из интернета. При необходимости его можно сохранить в data/workers_compensation.csv и модифицировать код для локальной загрузки.

    Для воспроизводимости результатов используется random_state=42.

    Приложение разработано в учебных целях и может быть доработано для реального использования.
